# Diabetes Risk Factor Analysis

Statistical analysis and visualization of diabetes prediction data using R to identify key health indicators and interpret predictive relationships.

## Project Overview

This project focuses on analyzing a diabetes prediction dataset to identify patterns and relationships between health-related variables and diabetes risk. Statistical analysis techniques and data visualization methods were used to explore the dataset, evaluate variable relationships, and interpret predictive trends.

The project was completed collaboratively as part of a statistical analysis coursework project using R.

## Technologies Used

- R
- ggplot2
- dplyr
- tidyverse
- Statistical Analysis
- Data Visualization

## Features

- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Statistical analysis of diabetes-related indicators
- Data visualization using graphs and plots
- Regression-based analysis
- Interpretation of predictive relationships
